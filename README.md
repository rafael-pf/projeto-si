# Como rodar

1. Clone o repositório:

```bash
git clone https://github.com/rafael-pf/projeto-si.git
cd projeto-si
```

2. Abra o projeto no VS Code.

3. Instale a extensão Live Server, se ainda não estiver instalada.

4. Inicie o Live Server:

- Clique com o botão direito no arquivo `index.html` e escolha "Open with Live Server".
- Ou use o atalho do Live Server na barra de status do VS Code.

5. O projeto será aberto no navegador automaticamente.

6. Para parar o servidor, feche a aba do navegador ou clique no botão de parada do Live Server no VS Code.
